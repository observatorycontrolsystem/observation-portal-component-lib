# observation-portal-component-lib

## Project requirements
-  Node.js version 8.9 or above (v10+ recommended).

## Project setup
```
npm install
```

### Compiles and minifies library for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints files and display lint errors
```
npm run lint:check
```

### Lints and fixes files
```
npm run lint:fix
```
